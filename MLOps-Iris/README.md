## ðŸ§  Dataset Information

The dataset used is the [Iris dataset](https://archive.ics.uci.edu/ml/datasets/iris), which contains 150 samples of iris flowers from three different species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

Each sample includes:
- Sepal length
- Sepal width
- Petal length
- Petal width
